# Игровая Механика

## Управление
- **WASD / Стрелки**: Перемещение игрока/камеры.
- **Мышь**: Обзор (если применимо) или взаимодействие с интерфейсом.

## Основной Цикл
1. Инициализация сцены.
2. Рендер кадров (60 FPS).
3. Обработка ввода пользователя.
4. Обновление состояния объектов.

## Игровые Объекты
- **Игрок**: Управляемый куб/персонаж.
- **Окружение**: Платформа, препятствия, декорации.
